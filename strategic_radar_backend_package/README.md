# Strategic Radar Backend Package

Repository-ready documentation pack for a governed multi-agent strategic radar backend.

## Structure
- `docs/01-strategy-and-scope/backend-blueprint.md`
- `docs/02-solution-architecture/solution-architecture.md`
- `docs/03-technical-spec/technical-spec.md`
- `openapi/openapi.yaml`

## Recommended use
- Blueprint: architecture intent, governance, scope, and control baseline
- Solution architecture: service boundaries, workflows, events, and delivery structure
- Technical specification: implementation contract
- OpenAPI: machine-readable API contract
- SQL DDL: initial PostgreSQL schema baseline
